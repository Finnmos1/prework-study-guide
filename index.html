Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Rel: specifies of the relationship between current document and linked document
  • href: location of external document

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • *: ALL

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all code into branch
  • git commit -m: Commit code from branch into repository
  • git push origin: push all changes into repository

JavaScript

  • If: Checks statement to see if TRUE/FALSE
  • For: Loops code inside until statment is FALSE
  • Function: lines of code that can be called together
